## jimage Crate Summary

Java module system `jimage` file reader for Java 9+ runtime images. Provides memory-mapped access to Java runtime modules for efficient class loading. Note: The jimage format is not officially documented.

For detailed conventions, module structure, and constraints, see `@/openspec/project.md`.
